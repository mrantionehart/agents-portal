=== DATABASE SCHEMA ===

  id: UUID (primary key)
  user_id: UUID (foreign key to auth.users)
  email: TEXT
  full_name: TEXT
  role: TEXT ('agent' | 'admin' | 'broker')
  agency_id: TEXT
  phone: TEXT
  created_at: TIMESTAMP
  updated_at: TIMESTAMP

Row Level Security Policies:
  - Users can view own profile

  - Admins can update all profiles
